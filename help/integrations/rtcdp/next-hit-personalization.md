# 使用Real-Time CDP和Adobe Target进行下一次点击个性化

了解如何使用Real-time Customer Data Platform和Adobe Target在下一次点击时个性化。 Real-Time CDP中的Adobe Target目标允许您在Adobe Target中使用Experience Platform区段来实现具有治理和隐私支持的相同页面和下一页个性化。
