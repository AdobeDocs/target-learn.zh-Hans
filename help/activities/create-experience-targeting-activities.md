# 创建体验定位活动

体验定位是迈入个性化世界的第一步，通常以 A/B 测试开始。营销人员运行 A/B 测试，并通过使用报表受众，发现不同的体验可与不同的受众产生共鸣。体验定位（或XT）允许营销人员轻松利用该发现，并将内容的长期定位切换到这些不同的用户区段。

## 描述

在此视频中，您将学习如何：
