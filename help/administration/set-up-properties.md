# 在[!DNL Adobe Target]中设置属性

## 描述

在本视频中，您将学习如何：

* 在[!DNL Adobe Target]中创建属性

## 目标受众

* 管理员管理员

## 要求

用户必须具有以下权限才能创建属性：

* 产品管理员（不要与产品配置文件管理员混淆）
* 系统管理员
