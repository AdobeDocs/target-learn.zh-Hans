# 使用用户档案比较在Adobe Target建立受众

Adobe Target的[!UICONTROL 用户档案比较]功能允许您在构建受众时，将两个数字用户档案值相互比较。 如果您要将自定义构建的倾向得分传递到[!DNL Target]并希望根据倾向得分最高的产品个性化内容，则此功能非常有用。
