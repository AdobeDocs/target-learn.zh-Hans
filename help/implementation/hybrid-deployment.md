# Adobe Target混合部署

Adobe Target优惠了一个独特的混合部署模型，用于个性化 — 混合了客户端和服务器端实施。 该混合模型允许非技术用户使用WYSIWYG Visual Experience Composer创作实验或个性化活动，并让服务器端执行、交付和渲染体验，以实现高性能投放。 

## 目标

在此视频的结尾，您将：

* 了解Adobe Target的混合部署
* 了解混合部署的优势
* 观察通过混合部署实现的服务器调用减少情况
