# 在Adobe Target中使用“配置文件比较”构建受众

Adobe Target中的[!UICONTROL 配置文件比较]功能允许您在构建受众时比较两个数字配置文件值。 例如，如果您将自定义生成的倾向分数传递到[!DNL Target]，并想要根据倾向分数最高的产品对内容进行个性化设置，则此功能非常有用。

## 您将了解的内容

在本视频中，您将学习如何：

* 比较传递到Adobe Target的数字配置文件值（分数）
* 使用这些得分比较来定义受众
