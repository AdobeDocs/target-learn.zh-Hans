# 在A/B测试和体验定位活动中使用Recommendations优惠

Recommendations优惠可与A/B测试和体验定位活动一起使用。 此功能打开了丰富功能，使您能够：

* 在同一活动内混合推荐和非推荐内容
* 轻松尝试在页面上放置推荐，包括多个推荐的顺序
* 使用自动分配功能自动将流量推送到效果最佳的推荐体验
* 使用自动目标，根据访客动态分配用户档案以定制推荐体验
